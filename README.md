# FairSplit - Expense Tracker

A PWA expense tracking app built with Next.js and Supabase.

## Setup

1. Copy `.env.local.example` to `.env.local` and add your Supabase credentials
2. Run `yarn install`
3. Run `yarn dev` to start development server

## Scripts

- `yarn dev` - Start development server
- `yarn build` - Build for production
- `yarn test` - Run tests with Vitest
- `yarn test:ui` - Run tests with UI
- `yarn test:coverage` - Run tests with coverage

## PWA Icons

Generate icons for PWA:
- Create a 512x512 icon and save as `public/icon-512.png`
- Create a 192x192 icon and save as `public/icon-192.png`

You can use tools like https://realfavicongenerator.net/ to generate these.
